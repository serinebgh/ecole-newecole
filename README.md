# New Ecole

Site web multi-pages d'une école de formation professionnelle, réalisé en HTML, CSS et JavaScript vanilla.

## Présentation

New Ecole est un projet web front-end présentant une école de formation fictive. Le site a été conçu avec une attention particulière portée au design, à la cohérence visuelle et à l'interactivité côté client.

---

## Pages

| Page | Fichier | Description |
|---|---|---|
| Accueil | `index.html` | Présentation de l'école et navigation principale |
| Formations | `formations.html` | Catalogue des formations, objectifs et débouchés |
| Inscription | `inscription.html` | Formulaire d'inscription avec calcul de devis dynamique |

---

## Fonctionnalités

- Design professionnel avec typographie Syne et Inter
- Header immersif avec dégradé bleu navy et effets visuels
- Cards de formation structurées avec niveau, durée et description
- Calcul de devis en temps réel (JavaScript vanilla)
  - Sélection de la formation avec prix unitaire associé
  - Option cours du soir avec majoration de 10 %
  - Prise en compte du nombre de participants
  - Résultat affiché instantanément dans une sidebar dédiée

## Installation

Aucune dépendance requise. Cloner le dépôt et ouvrir `index.html` directement dans un navigateur.

```bash
git clone https://github.com/serinebgh/ecole-newecole.git
cd ecole-newecole
```
